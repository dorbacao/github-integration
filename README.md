# Como criar um PAT (Personal Access Token)
Para gerar um token de acesso pessoal no GitHub, siga os passos abaixo:

Faça login na sua conta do GitHub.
No canto superior direito da página, clique na sua foto de perfil e selecione "Settings" (Configurações) no menu suspenso.
Na página de configurações, clique em "Developer settings" (Configurações de desenvolvedor) no menu lateral esquerdo.
No menu lateral esquerdo, clique em "Personal access tokens" (Tokens de acesso pessoal).
Na página "Personal access tokens", clique em "Generate new token" (Gerar novo token).
Na próxima página, forneça uma descrição para o token, para que você possa identificá-lo posteriormente.
Selecione as permissões adequadas para o token, dependendo das ações que você pretende realizar. Por exemplo, se você só precisa ler informações de repositórios públicos, selecione as permissões apropriadas. Se você precisar de acesso a repositórios privados ou realizar ações mais avançadas, selecione as permissões correspondentes.
Após selecionar as permissões, clique em "Generate token" (Gerar token).
O GitHub irá gerar um token de acesso pessoal para você. Copie o token e armazene-o em um local seguro, pois não será possível vê-lo novamente.
Use o token gerado substituindo 'seu_token_de_acesso_pessoal' no código que forneci anteriormente.
